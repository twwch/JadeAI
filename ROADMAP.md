# JadeAI Roadmap

## P0 - 高优先级

### 主题编辑器
- 数据库 schema 中已有 `themeConfig` 字段，但缺少编辑 UI
- 构建可视化面板，支持自定义颜色、字体、间距、行高
- 调整时实时预览效果
- 提供预设主题 + 自定义模式

### JD 匹配 AI 优化
- 粘贴目标职位描述（JD），AI 分析简历匹配度
- 关键词差距分析（缺失技能、行业术语）
- ATS 兼容性评分
- 一键生成针对特定职位的优化建议

### 设置页面
- UI Store 中已有 `settings` modal 骨架，待完善
- AI 模型配置（API Key、Base URL、模型选择）
- 默认语言偏好
- 主题偏好（浅色/深色/跟随系统）
- 自动保存开关及间隔
- 导出默认选项

### 更多简历模板
- ATS 友好型（简洁单栏、无装饰元素）
- 创意型（适合设计师，色块与布局更丰富）
- 学术/CV 型（支持发表论文、研究经历、教学经历等区块）
- 双栏布局
- 紧凑型（一页内容纳更多信息）
- 目标：总计 10+ 套模板

### AI 简历翻译
- 一键将整份简历在中英文之间互译
- 复用现有 i18n 基础设施
- 保留格式和区块结构
- 支持按区块单独翻译

## P1 - 中优先级

### AI 一键生成简历
- 输入：职位名称 + 工作年限，或 LinkedIn URL
- AI 自动生成完整简历草稿
- 用户在此基础上审阅和编辑

### AI 求职信生成
- 根据简历 + 目标 JD 自动生成求职信（Cover Letter）
- 多种语气可选（正式、亲和、自信）
- 模板化输出，支持导出

### AI 语法与措辞优化
- 自动检测拼写错误、弱动词、模糊描述
- 建议量化成果（如「负责销售」→「主导销售团队，业绩提升 30%」）
- 按行业推荐动作动词
- 要点（Bullet Point）优化

### 更多导出格式
- Word (.docx)
- HTML（独立页面）
- 纯文本
- JSON（数据迁移用）

### PDF 导出选项
- 页面尺寸（A4 / Letter）
- 页边距控制
- 水印开关
- 页眉/页脚自定义

### 仪表盘增强
- 简历搜索与筛选
- 标签/文件夹分组管理
- 按最近编辑/创建时间/名称排序
- 列表视图/网格视图切换

### 简历分享链接
- 为任意简历生成公开访问链接
- 可选密码保护
- 访问量统计
- 自定义短链接（如 `/r/john-doe-2024`）

## P2 - 锦上添花

### Anthropic Claude 原生支持
- 引入 `@ai-sdk/anthropic` 作为备选 Provider
- 在设置中切换 OpenAI / Claude 模型
- 支持按对话选择模型

### 版本历史
- 每次保存时记录编辑快照
- 时间线 UI 浏览历史版本
- 版本间差异对比视图
- 一键回滚到任意历史版本

### 键盘快捷键
- `Ctrl/Cmd + S` — 保存
- `Ctrl/Cmd + Z / Y` — 撤销 / 重做
- `Ctrl/Cmd + P` — 切换预览
- `Ctrl/Cmd + /` — 切换 AI 面板
- 快捷键速查弹窗

### 深色模式完善
- `next-themes` 已引入但可能未覆盖所有组件
- 全面审查并修复所有 UI 组件的深色模式适配
- 确保 PDF 预览在深色/浅色模式下一致

### 移动端适配
- 平板端响应式编辑器布局
- 移动端友好的简历查看器
- 移动端支持基础编辑（区块内容编辑，不含拖拽）

### 新手引导教程
- 首次使用的分步引导流程
- 高亮关键功能（AI 聊天、模板、导出）
- 可关闭，设置中可重新开启

### Markdown 编辑模式
- 为「个人总结」和「工作描述」等长文本提供富文本 Markdown 编辑器
- 编辑时实时预览
- 常用格式工具栏（加粗、列表、链接）

## P3 - 远期探索

### LinkedIn 导入
- 解析 LinkedIn 个人主页或导出数据
- 自动映射到简历各区块

### 简历对比
- 两份简历并排差异对比视图
- 高亮新增、删除、修改内容

### 简历评分看板
- 各区块完成度百分比
- 整体质量评分
- 改进建议
- AI 使用统计

### 模板市场
- 社区贡献模板
- 浏览、预览、安装
- 评分与下载量统计

### 实时协作编辑
- 基于 WebSocket / CRDT 的多人协同编辑
- 光标状态同步与实时更新
- 评论与建议模式

### 第三方集成
- Notion 导出
- Google Docs 同步
- ATS 平台 Webhook
